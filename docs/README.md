# docs/

このディレクトリにはプロジェクトのドキュメントを格納します。

## サブディレクトリ

- **analysis/**: 分析レポート
  - データ分析、特徴量分析、モデル分析のレポート
  - フォーマット: `YYYYMMDD_topic.md` または `topic_analysis.md`

- **strategies/**: 戦略ドキュメント
  - モデリング戦略、特徴量エンジニアリング戦略
  - コンペティション戦略、次のステップ

- **summaries/**: サマリー、まとめ
  - 実験のまとめ、最終サマリー
  - ベンチマーク結果、ベストモデルのまとめ

- **archive/**: 古いドキュメント
  - 使用しなくなったドキュメントを移動

## ルート直下のファイル

現在いくつかのドキュメントがルートディレクトリにありますが、以下のように整理することを推奨：

- 分析レポート系 → `analysis/`
- 戦略・計画系 → `strategies/`
- サマリー系 → `summaries/`
