# experiments/

このディレクトリには実験に関連するファイルを格納します。

## サブディレクトリ

- **logs/**: 実験ログ（Markdown形式）
  - フォーマット: `YYYYMMDD_experiment_name.md`
  - 実験の目的、仮説、結果、考察を記録

- **results/**: 実験結果（数値、テキスト）
  - フォーマット: `YYYYMMDD_experiment_name.txt`
  - スコア、評価指標、特徴量重要度などを記録

- **configs/**: 実験設定ファイル
  - JSON、YAML、Python設定ファイルなど
  - ハイパーパラメータ、特徴量セットの定義
